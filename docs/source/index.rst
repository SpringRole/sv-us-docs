SpringVerify-US API Developer's Guide 
=====================================

The SpringVerify-US API is organized around REST. Our API has predictable resource-oriented URLs, accepts form request JSON bodies, returns JSON responses, and uses standard HTTP response codes, authentication, and verbs.

.. note::
   This documentation is intended for developers only and not general users.
